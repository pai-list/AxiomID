# Reference Standards

This directory documents the technical standards, protocols, and specifications that AxiomID builds upon. Each page links to the official specification and describes how AxiomID implements it.

## W3C Standards

- [DID Core 1.0](w3c/did-core.md) — Decentralized Identifiers and the `did:axiom:` method
- [Verifiable Credentials Data Model 1.1](w3c/verifiable-credentials.md) — Credential issuance and verification
- [Data Integrity 1.0 (Ed25519)](w3c/data-integrity.md) — Cryptographic proof suites

## Stellar

- [Stellar Network](stellar/stellar-network.md) — Blockchain for hash anchoring and timestamps
- [Stellar Anchor Platform](stellar/stellar-anchor.md) — SEP standards for regulated asset issuance

## Pi Network

- [Pi SDK v2](pi-network/pi-sdk-v2.md) — Authentication, payments, native features
- [Pi Payments](pi-network/pi-payments.md) — Three-step payment flow (create → approve → complete)

## IPFS

- [IPFS Content Addressing](ipfs/ipfs-standards.md) — CIDs and distributed content storage

## ISO

- [ISO 27001:2022](iso/iso-27001.md) — Information security management framework

---

These references serve as the canonical source of truth for standard compliance during development, architecture decisions, and security reviews.
