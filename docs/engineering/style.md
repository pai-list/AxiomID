# 🎨 Style: Sovereign Aesthetic

## Design Mandate
- **Palette**: OLED Black (`#10131a`), Electric Blue (`#3b82f6`), Neon Emerald (`#22c55e`), Axiom Purple (`#6366f1`).
- **Feel**: Glassmorphism, `backdrop-blur`, subtle 1px borders.
- **Motion**: Smooth spring transitions using `framer-motion` easing `[0.16, 1, 0.3, 1]`.
- **Typography**: Geist Sans (Body), Geist Mono (Data/Code).
