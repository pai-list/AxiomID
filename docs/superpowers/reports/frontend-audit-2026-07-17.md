## 11. Recommendations

### Priority: High

1. **Add Suspense boundaries** for TanStack Query pages (explorer, leaderboard, status) to replace imperative loading flags with declarative Suspense.

2. **Server-side search/filter** on leaderboard — current client-side filtering doesn't scale past ~100 entries.

3. **Accessibility audit** for dialog components — verify focus trapping, Escape handling, and aria-modal usage across all `<dialog>` elements.

### Priority: Medium

4. **Add lazy loading for lucide-react icons** — consider dynamic icon imports for pages with heavy icon usage (dashboard, marketplace).

5. **Consolidate skeleton components** — 6 page-specific skeletons could be reduced to 2-3 with configurable layouts.

6. **Add test coverage** for SpendRequestsPanel (payment flow), TerminalOverlay (TUI patterns), and PassportKeyManager (crypto operations).

### Priority: Low

7. **Image optimization** — audit `<img>` tags for Next.js `<Image>` replacement opportunities.

8. **Context memoization** — verify WalletProvider and LanguageProvider aren't recreating objects on every render (no observed issues, but worth auditing).

9. **Docs page SSG** — docs home page is currently client-side static; could be SSG for better LCP.
