# T1.1 ‚Äî Create `src/lib/herenow.ts` (here.now API Client)

## Overview

Implement a here.now API client with 3-step publishing flow (create ‚Üí upload ‚Üí finalize) + convenience method, typed errors, and unit tests.

## Files to Create

### 1. `src/lib/herenow.ts`

**Exports:**
- `HereNowError` class (extends Error) with `code`, `statusCode` fields
- `createHereNowClient()` factory that returns the client object

**Client methods:**
- `createPage(opts: { title: string; slug: string }): Promise<{ id: string; url: string }>` ‚Äî POST `/pages`, timeout 5s
- `uploadContent(pageId: string, html: string): Promise<{ ok: boolean }>` ‚Äî PUT `/pages/:id/content`, timeout 30s
- `finalizePage(pageId: string): Promise<{ url: string; publishedAt: string }>` ‚Äî POST `/pages/:id/finalize`, timeout 5s
- `publishPage(opts: { title: string; slug: string; html: string }): Promise<{ url: string }>` ‚Äî calls create‚Üíupload‚Üífinalize in sequence

**Implementation details:**
- Read `HERENOW_TOKEN` and `HERENOW_API_URL` from `process.env`
- Base URL defaults to `https://api.here.now/v1`
- All requests use `Authorization: Bearer <token>` header
- Timeout via `AbortController` + `setTimeout`
- Non-OK responses parsed as JSON error body ‚Üí throw `HereNowError`
- Network/timeout errors wrapped in `HereNowError`
- Uses only native `fetch` (no deps)

### 2. `src/__tests__/lib/herenow.test.ts`

**Test cases (mock `global.fetch`):**
- `createPage` sends correct POST with auth header, returns id/url
- `createPage` throws `HereNowError` on non-2xx
- `uploadContent` sends PUT with html body
- `uploadContent` throws on failure
- `finalizePage` sends POST, returns url/publishedAt
- `publishPage` calls create‚Üíupload‚Üífinalize in sequence
- `publishPage` throws if any step fails
- Timeout handling (AbortError ‚Üí HereNowError)
- Missing token ‚Üí throws
- Custom base URL from env

### 3. `.env.example` update

Add `HERENOW_TOKEN` and `HERENOW_API_URL` entries.
