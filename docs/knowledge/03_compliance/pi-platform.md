## Requirement Detail

### Pi Browser Detection

- **Method:** 4-tier detection (any true = Pi Browser)
  1. `window.Pi` object exists (`src/lib/pi-sdk.ts:211`)
  2. User-Agent matches `/Pi Browser|minepi|PiApp/i` (`src/lib/pi-sdk.ts:213-214`)
  3. Iframed with `minepi.com` referrer (`src/lib/pi-sdk.ts:216-221`)
  4. Hostname is `minepi.com` (`src/lib/pi-sdk.ts:224-229`)
- **Coverage:** All known detection vectors covered
- **Fallback:** "Pi Browser Required" banner in ConnectStep (`src/components/claim/ConnectStep.tsx:76-88`)

### Pi SDK Initialization

- **Script source:** `https://sdk.minepi.com/pi-sdk.js` (async, no SRI — intentional `ponytail` at `src/lib/pi-sdk.ts:66`)
- **Init call:** `Pi.init({ version: "2.0", sandbox: determineSandboxMode() })`
- **Init guard:** Module-scoped `isInitialized` boolean — prevents double-init (`src/lib/pi-sdk.ts:33`)
- **Re-init:** On "not initialized" error, resets `isInitialized = false` and retries (`src/lib/pi-sdk.ts:345-348`)
- **Error recovery:** All PiSdkError codes defined (`src/lib/pi-sdk.ts:3-12`)

### Pi Auth Flow

1. Client: `connectPi()` → `loadPiSdk()` → `ensurePiInitialized()` → `Pi.authenticate(["username", "payments"], callback)`
2. Client receives: `{ user: { uid, username, name, stellarAddress }, accessToken }`
3. Client sends: `POST /api/auth/pi` with `{ accessToken, uid, username }`
4. Server: Zod validation → rate limiting → sandbox check → Pi API verification (`GET /api.minepi.com/v2/me`) → user upsert → return profile
5. Token stored encrypted: `piAccessToken` encrypted via `encryptToken()` (`src/app/api/auth/pi/route.ts:145`)
6. Token cached: `localStorage.setItem("pi_access_token", ...)`

### Pi Payment Flow

1. `createPiPayment(amount, memo, metadata)` → `window.Pi.createPayment()`
2. **onReadyForServerApproval:** `POST /api/pi/payment/approve`
3. **onReadyForServerCompletion:** `POST /api/pi/payment/complete`
4. **onError:** Reject with PiSdkError
5. **onCancel:** Reject with cancellation message
6. Auth token sent in `Authorization: Bearer` header from localStorage

### Sandbox vs Production

| Dimension | Sandbox | Production |
|-----------|---------|------------|
| Detection | `determineSandboxMode()` returns true | `determineSandboxMode()` returns false |
| Domain | localhost, 192.168.*, 10.*, staging | axiomid.app, *.axiomid.app |
| Token verification | `SANDBOX_DEV_TOKEN` bypass | `GET api.minepi.com/v2/me` |
| Auth middleware | Sandbox bypass enabled | Bypass blocked by `NODE_ENV` guard |
| Stellar | Hardcoded test address `GD5TJZ...` | Real address from Pi API response |
| Guard | `NODE_ENV !== "production"` at `src/app/api/auth/pi/route.ts:66` | No sandbox path executed |
