# ADR 0001: Transition to Axiom Identity Protocol (AIP)
**Status**: ACCEPTED
**Context**: Moving from a Pi-centric App to a Universal Identity Infrastructure.
**Decision**: 
- Pivot to a 3-layer architecture (Pi Anchor $\rightarrow$ Edge Orchestration $\rightarrow$ DX Interface).
- Implement Capability-Based Security.
- Transform Skills into Genomes.
**Consequences**: Requires significant refactoring of auth and marketplace logic.
