# 🌌 PAI Universe

> **People, Agents, Intelligent — Build Your Economy.**

PAI is the open ecosystem for agentic infrastructure built on Pi Network. We provide identity, commerce, and skill primitives for the next generation of AI agents.

## 🏗️ The Stack

```
Identity ← AxiomID (DID + TrustChain + Passport)
Gateway  ← pai-mcp (Unified MCP — CF/Vercel/Ghost/Pi)
CLI      ← pai-cli (One-command agent management)
Skills   ← pai-skills (Installable capability registry)
Agent Kit← pai-agent-kit (SDK + templates)
UI       ← pai-atom (Shared design primitives)
```

## 🚀 Quick Start

```bash
# Install the PAI CLI
npm install -g @pai/cli

# Create your first agent
pai agent create my-agent

# Deploy
pai deploy
```
