# @axiomid/langchain

Official LangChain/LangGraph integration for AxiomID — the Human Authorization Protocol for AI Agents.

## Features

- ✅ **DID Verification** — Verify `did:axiom` identities for agents
- ✅ **Soul Gate Enforcement** — Check ethical gates before task execution
- ✅ **Trust Score Retrieval** — Fetch and inject trust scores as context
- ✅ **Attestation Issuance** — Issue verifiable attestations from within agents
- ✅ **Streaming/Async Support** — Full async/await and streaming compatibility
- ✅ **Delegation Chain Demo** — A → B → C agent delegation with trust verification
